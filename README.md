
# Calculadora básica en Android

Esta es una aplicación simple de Android diseñada para realizar operaciones aritméticas básicas

## Características

- **Entrada de Números:** Dos campos de texto (`EditText`) para ingresar los números.
- **Operaciones Aritméticas:** Permite realizar suma y división de los números ingresados.
- **Manejo de Errores:** Alertas para entradas no válidas y división por cero.

## Cómo Funciona

1. **Entrada de Números:** El usuario introduce dos números en los campos proporcionados.
2. **Realizar Cálculo:**
   - Para sumar los números, el usuario presiona el botón de suma.
   - Para dividir, se utiliza el botón de división.
3. **Visualización de Resultados:** Los resultados se muestran mediante mensajes emergentes (`Toast`).

## Configuración del Proyecto

- **MainActivity:** Contiene la lógica principal de la aplicación.
- **activity_main.xml:** Define la interfaz de usuario con campos de texto y botones.
- **AndroidManifest.xml:** Configura los detalles de la aplicación y declara `MainActivity` como la actividad principal.

## Requisitos

- Android SDK
- Android Studio

## Instalación y Ejecución

1. Clona o descarga este repositorio.
2. Abre el proyecto en Android Studio.
3. Ejecuta la aplicación en un emulador o dispositivo Android.
